# 보안 및 개인정보 가이드

> AI 도구를 안전하게 사용하기 위해 꼭 알아야 할 원칙들을 정리했습니다.

---

## 공통 원칙

AI 도구의 종류나 버전은 계속 바뀌지만, **보안 원칙은 동일합니다.**

!!! warning "핵심 원칙"
    **AI에게 입력하는 모든 내용은 "누군가 볼 수 있다"고 가정하세요.**

    아무리 안전한 시스템이라도, 입력 단계에서 민감한 정보를 넣지 않는 것이 가장 확실한 보호 방법입니다.

### 절대 입력하면 안 되는 정보

| 구분 | 예시 |
|------|------|
| 개인 식별 정보 | 주민등록번호, 여권번호, 운전면허번호 |
| 금융 정보 | 카드번호, 계좌번호, 비밀번호 |
| 인증 정보 | 로그인 비밀번호, API 키, 인증 토큰 |
| 타인의 개인정보 | 다른 사람의 이름+연락처 조합, 사진 속 얼굴 |

## 임직원(비개발자) — 사내 엔터프라이즈 환경

### 엔터프라이즈 AI는 안전한가요?

회사에서 제공하는 AI 도구는 **엔터프라이즈 계약**을 통해 운영됩니다. 이는 다음을 의미합니다.

- **데이터 학습 제외**: 입력한 내용이 AI 모델 학습에 사용되지 않습니다
- **데이터 격리**: 우리 회사의 데이터는 다른 회사와 분리되어 처리됩니다
- **접근 통제**: 회사 계정으로만 접근 가능하며, 사용 이력이 관리됩니다

### 그래도 주의할 점

엔터프라이즈 환경이라도 다음 사항은 지켜야 합니다.

1. **대외비·기밀 문서는 확인 후 사용**: 문서의 보안 등급을 확인하세요. 사내 AI 사용이 허용된 등급인지 보안 정책을 따릅니다.
2. **개인정보가 포함된 데이터는 가리거나 빼고 입력**: 고객 데이터, 직원 정보 등은 이름·연락처 등을 제거하거나 가린 뒤 입력하세요. (전문 용어로는 "비식별 처리"라고 합니다.)
3. **결과물의 외부 공유 주의**: AI가 생성한 결과물에 사내 정보가 포함될 수 있습니다. 외부에 공유하기 전에 반드시 검토하세요.

!!! tip "한 줄 요약"
    사내 AI는 안전하게 설계되어 있지만, **"입력 전에 한 번 더 생각하기"** 습관은 여전히 중요합니다.

## 학생 / 일반인 (무료 AI 도구)

### 무료 AI 도구의 특성

무료로 제공되는 AI 도구(ChatGPT 무료 버전, Gemini 등)는 엔터프라이즈 환경과 다릅니다.

- **입력 내용이 서비스 개선에 활용될 수 있습니다** (약관에 따라 다름)
- **대화 내용이 서버에 저장될 수 있습니다**
- **보안 수준이 유료/기업용보다 낮을 수 있습니다**

### 꼭 지켜야 할 수칙

#### 1. 내 개인정보를 넣지 않기

- 이름, 전화번호, 주소, 학번 등을 입력하지 마세요
- "내 이름은 OOO이고 OO학교 다녀"처럼 자연스럽게 노출되는 경우도 주의하세요

#### 2. 다른 사람의 정보를 넣지 않기

- 친구, 가족, 선생님 등 타인의 이름·연락처를 입력하지 마세요
- 단체 사진이나 다른 사람이 찍힌 사진을 업로드하지 마세요

#### 3. 사진 속 개인정보 확인하기

- AI에 사진을 업로드할 때, 사진 속에 **이름표, 학생증, 배경의 주소판** 등이 보이지 않는지 확인하세요
- 문서 사진의 경우 민감한 부분을 가리고 업로드하세요

#### 4. 결과물을 그대로 믿지 않기

- AI가 생성한 정보가 항상 정확한 것은 아닙니다
- 특히 사실 확인이 중요한 내용(뉴스, 통계, 법률 등)은 반드시 원본 출처를 확인하세요

!!! tip "한 줄 요약"
    무료 AI 도구는 편리하지만, **"나와 다른 사람의 개인정보는 절대 입력하지 않기"**를 기억하세요.
