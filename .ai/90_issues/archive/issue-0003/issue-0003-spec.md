# Issue #0003 스펙 비개발자 대상 AI 활용 교육자료 - 발표용 슬라이드 화면 추가

## 목표 (Goal)

기존 MkDocs Material 문서 사이트에 슬라이드 모드 전환 기능을 추가하여, 동일한 마크다운 콘텐츠를 발표자료로도 활용할 수 있게 한다.

---

## 범위 (Scope)

**포함 (In)**

- 슬라이드 모드 전환 버튼 및 오버레이 UI 구현
- h2/h3 단위 슬라이드 자동 분할
- 슬라이드 전용 시각 디자인 (다크 테마, 그라데이션, 카드 레이아웃)
- 페이지 점프 바, 하단 네비게이션 바
- 콘텐츠 유형별 슬라이드 스타일 (카드 그리드, 스텝 카드, 체크리스트, admonition 강조)

**비포함 (Out)**

- 마크다운 원본 수정
- PDF 내보내기 기능
- reveal.js 외부 라이브러리 의존성 추가

---
