#### Task 14: "참가자 준비 / 강사 준비"를 좌우 대칭 + 비대칭 강조 admonition 2개로 정렬

- **결과**: 완료 (4차 반복으로 정착)
- **수행 내용 요약**: docs(`index.md:94-105`)는 `<div class="grid" markdown>` 안에 `!!! info "참가자가 준비"` / `!!! note "강사가 준비 (참가자는 신경 쓰지 않아도 됩니다)"` 두 개의 admonition으로 교체 — Material에서 `info`는 블루, `note`는 회색. slides(`slides.md:238-260`)는 `<div grid="~ cols-2 gap-4">` 안에 참가자=blue-toned(`border-blue-400 + bg-blue-50/40 dark:bg-blue-900/15`), 강사=gray-toned(`border-gray-400 + bg-gray-50/40 dark:bg-gray-700/20`) admonition 박스 좌우 배치. 이모지 제거(텍스트 라벨만), 콘텐츠 항목은 유지(Task 13의 Claude Desktop 포함).
- **특이 사항**:
  - **반복 이력 (디자인 합의 과정)**:
    1. 1차: ✅/❌ 카드 패턴 그대로 적용 → 사용자 지적 "거의 동일 맥락인데 카드 남용 아닌가"
    2. 2차: 단일 info admonition + 🎒/👨‍🏫 이모지 라벨로 묶음 → 사용자 지적 "좌우 박스 형태가 더 나을 듯 + 이모지 빼"
    3. 3차: 좌우 대칭 info admonition 2개(같은 톤), 이모지 제외 → 사용자 지적 "강사 쪽은 회색톤이 좋을 것 같아"
    4. 4차(현재): 좌우 대칭 유지, 비대칭 강조 — 참가자=info 블루(액션 아이템), 강사=note 회색(안내 정보)
  - **합의된 시각 어휘 원칙 (정련됨)**:
    - 본질적 대비/대립 → 색이 다른 카드 박스 (대비 강조)
    - 동등 분담 → 같은 톤 좌우 대칭 admonition (가짜 갈등 회피)
    - **무게가 다른 분담** → 좌우 대칭 + 비대칭 톤(주목 톤 + 무채 톤)으로 의미 위계 표현 — Task 14에서 새로 정착한 어휘
    - 단순 나열 회피
  - **이모지 미사용 룰**: 사용자가 "이모지 넣지마" 명시 — 이번 슬라이드/문서 라벨링에서 텍스트 라벨이 차분 톤과 더 잘 맞음을 확인. 이후 비슷한 contexts에서도 이모지 신중 사용.
