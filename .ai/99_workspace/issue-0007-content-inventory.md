## 3. 페이지별 시각 요소 사용 현황

### 3.1 `index.md` (131줄)

| 요소 | 개수 | 위치/비고 |
|------|------|---------|
| H2 섹션 | 5 | Why / Who / What / How / 함께 읽어보세요 |
| 표 | 2 | "대상자 프로필"(2×2), "결과물 예시"(2×2) |
| admonition(info) | 2 | "공통 전제"(Who), "실습 페이지 안내"(How 끝) |
| 순서 없는 리스트 | 다수 | Why 본문, 요구/비요구 지식, 준비사항 체크리스트 |
| 체크박스 리스트 | 1블록 | "참가자가 준비할 것" 5항목 |
| 이미지 | 0 | — |
| Mermaid/차트 | 0 | — |
| blockquote | 2 | 도입문, "핵심은…" 강조 |

**텍스트 비중 메모**
- **Why 섹션**이 가장 텍스트 위주 — 3단계 가치(기능/시간/비용)와 "AI가 바꾸는 것" 3항목이 모두 서술형 bullet.
  → **아이콘 카드 혹은 비교표**로 시각화 유효.
- **How 섹션 "실습 접근법 4단계"**(문제정의→최소플랜→실행→개선)가 번호 리스트로만 표현.
  → **Mermaid flowchart / 단계 박스**로 치환 유효.
- **Who 섹션 사전지식(요구/비요구)** 두 bullet 그룹 → **2열 비교표**가 더 직관적.

**텍스트 비중 메모**
- 이미 표·체크박스가 많아 상대적으로 시각화 비중 양호.

## 4. 전체 관찰 요약

|------|------|
| 이미지 자산 0개 | `docs/assets/images/`가 비어있어 스크린샷/아이콘 삽입 여지가 큼 |
