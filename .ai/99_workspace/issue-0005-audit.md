## 4. Task 2 진입 시 결정 사항

1. **Who / 사전지식 / 준비사항**을 index.md 하나의 큰 섹션 아래 하위 제목으로 묶을지, 병렬 3섹션으로 둘지
   → 권장: **"Who — 누구를 위한 교육인가요?"** 대제목 아래 "대상자 / 사전지식 / 준비사항" 3개 하위 제목. 이유: 세 항목 모두 "내가 이 교육에 맞는 사람인가?"라는 동일 질문에 답하므로 묶는 쪽이 흐름상 자연스러움.

2. **트랙별 분리 방식**
   → Who(대상자)·준비사항은 표로 임직원/학생·일반인 2열 분리. 사전지식은 두 트랙 공통이므로 단일 표.

3. **"경험 없음" 참가자 처리 (C1)**
   → 본 이슈에서는 index.md 기준으로 "챗봇형 AI 사용 경험이 전제"임을 명시. operation-guide.md의 "경험 없는 그룹" 문구는 Task 3에서 "사전 학습 후 참가" 또는 선택지 축소로 정리.
